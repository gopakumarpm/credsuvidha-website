Logo

The CredSuvidha logo features interlocking C and S letterforms with an upward growth arrow, symbolizing financial progress and partnership.

Primary — Light Background
Inverted — Dark Background
On Brand Cream — #FAF6F1

Logo Elements

Symbol: Interlocking "C" (navy blue) and "S" (gold) with upward arrow
Wordmark: "CREDSUVIDHA.COM" in navy blue uppercase
Tagline: "TRUSTED PARTNER. SWIFT SOLUTIONS." in gold

Clear Space & Sizing

Maintain minimum clear space equal to the height of the "C" around all sides of the logo. Minimum display size: 120px wide for digital, 30mm for print.

✓ Do

Use on white, cream, or very light backgrounds
Use the inverted (white) version on dark backgrounds
Maintain proportions when scaling

✗ Don't

Don't stretch or distort the logo
Don't change the logo colors
Don't place on busy or low-contrast backgrounds
Don't add effects like drop shadows or outlines

Buttons & UI Components

Glass Morphism

Glass card effect used on hero stats and overlaid content

Card Hover

Cards elevate on hover with translateY(-8px) and box-shadow: 0 25px 60px rgba(0,0,0,0.12). Transition uses cubic-bezier(0.4, 0, 0.2, 1) for smooth feel.

Technical Specifications

Technology stack and external dependencies for the website.

CSS Framework

Tailwind CSS via CDN (cdn.tailwindcss.com) with custom theme configuration for brand colors, fonts, and extended utilities.

Fonts

Google Fonts CDN: Inter (300-900) and Playfair Display (700-800). Preconnected for performance.

JavaScript

Vanilla JavaScript. No external libraries. Features: scroll reveal (Intersection Observer), counter animation, mobile menu, smooth scroll.

Hosting

Netlify (static hosting). Custom domain: www.credsuvidha.com. Single HTML file deployment. No build step required.
